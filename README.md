# Peaks

## Install audiowaveform

Ubuntu
```
sudo add-apt-repository ppa:chris-needham/ppa
sudo apt-get update
sudo apt-get install audiowaveform
```

MacOs
```
brew tap bbc/audiowaveform
brew install audiowaveform
```

# Install ffmpeg

Ubuntu
```
sudo apt-get update
sudo apt install ffmpeg
```

Macos
```
brew install ffmpeg
```
